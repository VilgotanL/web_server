brainf**k

Brainf**k is an esoteric programming language (esolang) where there are only 8 instructions all of which are a single character long,
all other characters are comments and they are ignored.

In brainf**k there is a tape going right that consists of bytes that can hold numbers between 0 to 255, initially 0,
there is also a pointer that starts on the leftmost cell of the tape.

0 0 0 0 0 ...
^

You can move the tape pointer left and right with > and <,
and you can increment (+1) and decrement (-1) the current cell with + and -.
When you increment past 255 the cell overflows back to 0, and vice-versa for decrementing.

You can print the current cell's value as an ascii character with the dot (.),
and you can input an ascii character with the comma (,).

The two last instructions are [ and ], they combined act as a while-not-zero loop,
[code] runs while the current cell is not zero.

Examples of []:
[-] set the current cell to zero
[->+<] move the current cell's value to the next cell

Edge Cases

If the pointer goes left of the leftmost cell, the behaviour is undefined.
If [ and ] are unbalanced or used in an invalid way, the behaviour is undefined.
If the end of input (aka eof) is reached the most common thing for an implementation is to leave the cell as-is or set it to 0, some implementations set it to -1.
